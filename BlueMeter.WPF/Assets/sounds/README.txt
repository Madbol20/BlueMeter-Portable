Queue Pop Alert Sound Files
============================

Place the following sound files in this directory:

Required Files:
- drum.mp3    - Drum sound for queue pop alert
- harp.mp3    - Harp sound for queue pop alert
- wow.mp3     - Wow sound for queue pop alert
- yoooo.mp3   - Yoooo sound for queue pop alert

If sound files are not found, the system will fall back to Windows system sounds.

You can use any MP3 files you like, just rename them to match the names above.

Recommended sources for free sounds:
- https://freesound.org/
- https://mixkit.co/free-sound-effects/
- https://pixabay.com/sound-effects/
